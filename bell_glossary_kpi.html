Виручка (Вир)
выр пл1
план виручки
Планова виручка поточного периоду
выр1
факт виручки (поточний)
Фактична виручка поточного звітного периоду
выр2
факт виручки (попередній)
Фактична виручка попереднього звітного периоду
вырΔ
зміна виручки абс.
Абсолютна зміна виручки: поточний vs попередній
вырΔ = выр1 − выр2
выр%
динаміка виручки %
Відсоткова зміна виручки: поточний vs попередній
выр% = выр1 / выр2 − 1
КВП Выр1
виконання плану виручки
Коефіцієнт виконання плану по виручці
КВП Выр = выр1 / выр пл1
Уд.вес Выр
частка виручки
Частка ТТ у загальній виручці компанії
Уд.вес Выр = выр1 (ТТ) / Σ выр1 (всі)
Валовий прибуток (ВП)
вп пл1
план ВП
Плановий валовий прибуток (виручка мінус собівартість)
вп1
факт ВП (поточний)
Фактичний валовий прибуток поточного периоду
вп2
факт ВП (попередній)
Фактичний валовий прибуток попереднього периоду
впΔ
зміна ВП абс.
Абсолютна зміна валового прибутку
впΔ = вп1 − вп2
вп%
динаміка ВП %
Відсоткова зміна валового прибутку
вп% = вп1 / вп2 − 1
КВП вп1
виконання плану ВП
Коефіцієнт виконання плану по валовому прибутку
КВП вп = вп1 / вп пл1
Уд.вес вп1
частка ВП
Частка ТТ у загальному валовому прибутку компанії
Уд.вес вп = вп1 (ТТ) / Σ вп1 (всі)
ROM — рентабельність по валовому прибутку
ROMпл
план ROM
Планова рентабельність
ROMпл = вп пл1 / выр пл1
ROM1
факт ROM (поточний)
Фактична рентабельність поточного периоду
ROM1 = вп1 / выр1
ROM2
факт ROM (попередній)
Фактична рентабельність попереднього периоду
ROM2 = вп2 / выр2
ROM%
зміна ROM в п.п.
Зміна рентабельності: поточний vs попередній
ROM% = ROM1 − ROM2
Трафік / Вхідний потік (кількість відвідувачів)
ТРпл1
план трафіку
Планова кількість відвідувачів за период
Тр1
факт трафіку (поточний)
Фактична кількість відвідувачів поточного периоду
Тр2
факт трафіку (попередній)
Фактична кількість відвідувачів попереднього периоду
Тр Δ
зміна трафіку абс.
Абсолютна зміна трафіку: поточний vs попередній
ТрΔ = Тр1 − Тр2
ΔТр%
динаміка трафіку %
Відсоткова зміна трафіку
ΔТр% = Тр1 / Тр2 − 1
Уд вес Тр
частка трафіку
Частка ТТ у загальному трафіку мережі
Уд вес Тр = Тр1 (ТТ) / Σ Тр1 (всі ТТ)
Конверсія
Конвпл1
план конверсії
Планова конверсія — частка відвідувачів, що зробили покупку
Конв1
факт конверсії (поточний)
Фактична конверсія поточного периоду
Конв1 = кількість транзакцій / Тр1
Конв2
факт конверсії (попередній)
Фактична конверсія попереднього периоду
Конв2 = транзакції попереднього / Тр2
Конв_Δ
зміна конверсії абс.
Абсолютна зміна конверсії в процентних пунктах
Конв_Δ = Конв1 − Конв2
ΔКонв%
динаміка конверсії %
Відсоткова зміна конверсії (не п.п., а %)
ΔКонв% = Конв1 / Конв2 − 1
Середній чек (СРчек)
СРчекпл1
план середнього чека
Плановий середній чек на одну транзакцію
СРчек1
факт середнього чека (поточний)
Фактичний середній чек поточного периоду
СРчек1 = выр1 / (Тр1 × Конв1)
СРчек2
факт середнього чека (попередній)
Фактичний середній чек попереднього периоду
СРчек2 = выр2 / (Тр2 × Конв2)
СРчекΔ
зміна середнього чека абс.
Абсолютна зміна середнього чека
СРчекΔ = СРчек1 − СРчек2
ΔСРчек%
динаміка середнього чека %
Відсоткова зміна середнього чека
ΔСРчек% = СРчек1 / СРчек2 − 1
ПНов — прибуток на одного відвідувача (PPV / NPV)
ПНов пл1
план PPV
Плановий прибуток на одного відвідувача
ПНов пл1 = вп пл1 / ТРпл1
ПНов1
PPV — поточний
Фактичний прибуток на одного відвідувача поточного периоду
ПНов1 = вп1 / Тр1
ПНов2
NPV — попередній
Фактичний прибуток на одного відвідувача попереднього периоду
ПНов2 = вп2 / Тр2
ΔПНов
зміна PPV абс.
Абсолютна зміна прибутку на відвідувача
ΔПНов = ПНов1 − ПНов2
ΔПНов%
динаміка PPV %
Відсоткова зміна прибутку на відвідувача
ΔПНов% = ПНов1 / ПНов2 − 1
ОбТО — оборотність товарних запасів
ОбТО1
оборотність ТЗ (поточний)
Оборотність товарних запасів поточного периоду
ОбТО1 = Собівартість продажів / Середні товарні залишки
Ключова формула виручки Виручка = Трафік × Конверсія × Середній чек
выр1 = Тр1 × Конв1 × СРчек1

Будь-яке відхилення виручки пояснюється через один або комбінацію трьох факторів: трафік, конверсія, чек.
